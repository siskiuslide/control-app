About


We provide an integration for Hubitat users who need a simple and snappy alternative to the built in Hubitat Dashboard. Connect to any number of networks by configuring instances of the MakerAPI to control devices you assign.

The project is built with NodeJS, Express & MongoDB, and utilizes a simplistic HTML/CSS/JS frontend.

Multiple connections

Allows connections to any number of Maker API instances setup on any hub with an internet connection. Use this to segment areas of your network or control multiple networks from a remote location.

Polling

Your networks can be configured to automatically make poll requests to your hub which will keep their status' up to date within the UI. Polling is toggleable and individual to each network.

Local or Cloud

Establish connections using local network addressing or control devices remotely using the cloud.

Documentation for installing this application for your local network can be found here

Instant response

The UI responds instantly to controlling devices and is followed by a validation check, ensuring the device's shown status is correct. Enjoy a snappy and reliable UI.
